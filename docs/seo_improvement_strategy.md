## ON-PAGE SEO CHECKLIST

### For Every Page:

```
□ Title tag (under 60 characters, includes primary keyword)
□ Meta description (150-160 characters, includes CTA)
□ H1 heading (one per page, includes keyword)
□ H2 subheadings (use secondary keywords)
□ Image alt text (descriptive, includes keywords naturally)
□ Internal links (link to related products/pages)
□ URL slug (short, includes keyword, no stop words)
□ Schema markup (appropriate type for page)
```

### Product Page SEO Template:

```
TITLE: [Product Name] | [Primary Keyword] - Himalayan Fibres
       Example: Fine Nettle Yarn | Organic Handspun Yarn Nepal - Himalayan Fibres

META DESCRIPTION: [Benefit] + [Product] + [Differentiator] + [CTA]
       Example: Premium organic nettle yarn handspun in Nepal. Sustainable,
       eco-friendly fiber perfect for knitting & weaving. Shop now or request samples.

H1: [Product Name with Keyword]
    Example: Fine Nettle Yarn - Organic Handspun Yarn from Nepal

CONTENT STRUCTURE:
- Opening paragraph (100 words, include primary keyword)
- Product features/specifications
- Benefits section
- How to use / Applications
- FAQ section (3-5 questions)
- Related products (internal links)
```

## LINK BUILDING STRATEGY

### Internal Linking:

Create a hub-and-spoke model:

```
                    HOMEPAGE
                       │
        ┌──────────────┼──────────────┐
        │              │              │
     SHOP           ABOUT         BLOG
        │              │              │
   ┌────┼────┐         │         ┌───┼───┐
   │    │    │         │         │   │   │
Product Product    Sustainability  Post Post
   1     2                        1    2
   │     │                        │    │
   └──────────────────────────────┴────┘
         (Cross-link related content)
```

**Internal Linking Rules:**
1. Every product links to 2-3 related products
2. Every blog post links to relevant products
3. Every product has link to category/shop page
4. Use keyword-rich anchor text (not "click here")

## GEO (Generative Engine Optimization)

### What is GEO?
Optimizing for AI search engines (ChatGPT, Perplexity, etc.)

### GEO Tactics:

| Tactic | Implementation |
|--------|----------------|
| **Statistics & Data** | Include verifiable numbers in content |
| **Citations** | Reference studies, research |
| **Comparisons** | Create comparison tables |
| **Direct Answers** | Answer questions clearly in first paragraph |
| **Structured Content** | Use clear headings, lists, tables |
| **Unique Information** | Provide information not found elsewhere |

### Example GEO-Optimized Content:

```
Q: What is Himalayan nettle fiber?

Himalayan nettle fiber comes from Girardinia diversifolia, a species of
stinging nettle native to the Himalayas (1,500-3,000m altitude). Studies
show nettle fiber has tensile strength comparable to hemp (600-700 MPa)
while being 2x stronger than cotton. The fiber is naturally antimicrobial
and biodegrades within 2 years.

Key statistics:
- Grows at 1,500-3,000m altitude
- Water requirement: 80% less than cotton
- Biodegradation time: 1-2 years
- No pesticides or fertilizers needed

[Source: Research papers, if available]
```

## COMMON SEO MISTAKES TO AVOID

1. ❌ **Keyword stuffing** - Don't overuse keywords unnaturally
2. ❌ **Duplicate content** - Each page should have unique content
3. ❌ **Ignoring mobile** - Always optimize for mobile first
4. ❌ **Slow images** - Always compress before uploading
5. ❌ **No internal links** - Link between related content
6. ❌ **Ignoring Search Console** - Check weekly for issues
7. ❌ **Expecting instant results** - SEO takes 3-6 months
