# HIMALAYAN FIBRES - Website Improvement Task Sheet

## Overview

This task sheet breaks down the website improvement recommendations into **4 phases** based on:
- **Impact**: How much it affects SEO/conversions
- **Effort**: Time and resources required
- **Dependencies**: What needs to be done first

## PHASE 1: Foundation & Quick Wins (Week 1-2)
**Goal**: Fix critical technical SEO issues and establish baseline visibility

### 1.1 Technical SEO - Critical Fixes
| Task | Priority | Effort | Impact | Status |
|------|----------|--------|--------|--------|
| Optimize meta titles for all 4 indexed pages (under 60 chars, include keywords) | HIGH | Low | Direct ranking factor | ⬜ |
| Write compelling meta descriptions (150-160 chars) for all pages | HIGH | Low | Improves CTR | ⬜ |
| Add alt text to ALL product images | HIGH | Low | Image search + accessibility | ⬜ |
| Create and submit XML sitemap to Google Search Console | HIGH | Low | Ensures indexing | ⬜ |
| Optimize URL slugs (cleaner URLs without `/product-page/` prefix) | HIGH | Medium | Better rankings | ⬜ |

### 1.2 Essential Pages to Create
| Page | Purpose | Priority | Status |
|------|---------|----------|--------|
| `/about` | Company story, mission, founder journey | HIGH | ⬜ |
| `/contact` (if not exists) | Contact info, WhatsApp, physical address | HIGH | ⬜ |
| `/shipping-returns` | Clear policies | MEDIUM | ⬜ |

### 1.3 Trust Signals (Quick Additions)
| Task | Location | Status |
|------|----------|--------|
| Add physical address prominently | Footer/Contact | ⬜ |
| Add WhatsApp/phone contact option | Header/Footer | ⬜ |
| Add secure payment badges | Checkout/Footer | ⬜ |
